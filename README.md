MNT Tamil RAG Chatbot

This project implements a Tamil-language Retrieval-Augmented Generation (RAG) ingestion and retrieval pipeline.
It converts a scanned Tamil PDF into structured, searchable knowledge using OCR, text preprocessing, semantic chunking, and FAISS-based vector search.

The project focuses on data ingestion and retrieval, which is the foundation of a RAG chatbot.
